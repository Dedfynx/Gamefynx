# Gamefynx

A multi-emulator project built in C++ with imGui and SDL3.


### Supported Systems

| System | Status | Notes |
|--------|--------|-------|
| CHIP-8 |  In Progress | Simple 8-bit system |
| Game Boy |  Planned | DMG (original Game Boy) |
